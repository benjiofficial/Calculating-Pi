# Calculating-Pi
Calculating pi using the square AGM method <br>
## Compiling
Dependacies: GMP, g++

GMP can be installed from <a href="https://gmplib.org/#DOWNLOAD">here</a>.
``` bash
$g++ Main.cpp -lgmp -o Pi-Calculator
$./Pi-Calculator
```
